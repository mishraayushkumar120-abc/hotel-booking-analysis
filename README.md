# hotel-booking-analysis
A comprehensive Hotel Booking Data Analytics project featuring Excel dashboards, pivot tables, CSV datasets, SQL queries, and Python-based exploratory data analysis. This project analyzes customer behavior, booking trends, cancellations, and revenue patterns to generate actionable insights for the hospitality industry.
Project Components
1. Excel Dashboard
Designed 2 dynamic dashboards
Visualized key metrics such as:
Booking trends
Customer segmentation
Cancellation rate
Revenue distribution
Seasonal patterns
Includes pivot tables, charts, slicers, and KPI cards
2. Dataset (CSV)
Original raw dataset
Cleaned/processed dataset
Used for all analysis steps
3. SQL Querie
Performed data cleaning
Extracted meaningful metrics:
Most booked hotel type
Highest cancellation months
Average stay duration
Most common customer segments
Query file included (queries.sql)
4. Python Notebook
Exploratory Data Analysis (EDA)
Data visualization using Matplotlib/Seaborn
Feature analysis
Outlier detection and trend analysis
Notebook included (analysis.ipynb)
